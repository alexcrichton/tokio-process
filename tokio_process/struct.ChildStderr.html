pub struct ChildStderr { /* fields omitted */ }

The standard error stream for spawned children.

This type implements the Read trait to read data from the stderr handle of a child process. Note that this type is also "futures aware" meaning that it is both (a) nonblocking and (b) will panic if used off of a future's task.
